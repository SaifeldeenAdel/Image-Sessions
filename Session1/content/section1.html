OpenCV - Working with Images

Loading and Displaying an Image

To load an image, use the imread function in OpenCV.

Displaying an image in OpenCV is done using the imshow function or by using Matplotlib. The second paramter being a 0 specifies greyscale.
Try removing the cmap arg from matplotlib and run the cell again.

How images are represented

We can see the following features in an image after loading it:
  • Images are stored as numpy arrays
  • 2D Matrix (Color images will be 3D)
  • Data type of uint8 (so all integer values will be in the range of 0 - 255)

Pixel Values explained

  • An image will have X*Y pixels and thats the shape of the 2D matrix stored.
  • Each pixel is a number value in the matrix and the numbers indicate the pixel's color

By creating a pixel array that goes from 0 to 255 and displaying it, we see that it produces a square that goes from black to white.
Meaning 0 = black , 255 = white

Binary images

Images that contain pixels with the values 0 or 255 only are called binary images.
