Mouse Events and callbacks

Images displayed using OpenCV can have a mouse callback function attached to them

What's a callback function?

It is a function that runs as a result of a certain event occuring. In our case, when we attach a mouse callback function to an image, whenever a mouse event occurs (click, double click ,etc), this function runs.
In openCV this function takes in a lot of paramters, most importantly though are the event and the x ,y coordinates. You can learn more about how a callback function works in our notebook and here

Drawing on our images

Often times we need to draw shapes on our images. Whether its a bounding box to detect a certain object or its some lines to detect edges or straight lines for example.
These can be done as a result of a mouse event or as part of a detection algorithm.
In OpenCV, there are built in functions for drawing over your images. For example if you need to draw a rectangle, you might find these results helpful.
